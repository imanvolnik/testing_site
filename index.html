Jupyter

Jupyter Notebook is an open-source tool that allows you to create and share documents with live code, equations, visualizations, and text. It's widely used in Data Science for interactive coding and data analysis.

Alex The Analyst - Jupyter Notebooks/Anaconda | Python for Beginners
Rob Mulla - Jupyter Notebook Complete Beginner Guide

SQL

SQL (Structured Query Language) is a standard language used to manage and manipulate databases. It's essential for querying, updating, and organizing data in relational databases, making it a key tool in Data Science and data management.

EngineerSpock - PostgreSQL course on YouTube

Statistics

Statistics is the branch of mathematics that deals with collecting, analyzing, interpreting, and presenting data. It's fundamental in Data Science for making data-driven decisions, identifying patterns, and drawing meaningful insights from datasets.

Business Intelligence (BI)

Business Intelligence (BI) tools like Power BI and Tableau are designed to analyze and visualize data, helping organizations make informed decisions. These platforms allow users to create interactive dashboards and reports, enabling real-time insights into business performance and trends. They simplify data exploration and enhance reporting capabilities, making data accessible to users at all levels.

Excel

Excel is a versatile spreadsheet application used for data analysis, organization, and visualization. It allows users to perform calculations, create charts, and manage datasets, making it essential for both simple and complex data tasks.
